Autonomous Drone Navigation & Control

University of Windsor - Elevate 2025

This project explored the development of autonomous flight systems using MATLAB and Simulink. I focused on modeling drone dynamics and implementing PID-based stabilization algorithms to manage thrust, lift, and torque for precise 3D maneuvering[cite: 10, 15, 37].

Key Technical Contributions:

  • Control Systems: Designed and tuned PID algorithms to calculate motor commands based on sensor feedback from IMUs, cameras, and pressure sensors[cite: 11, 16, 25].
  • Software Integration: Utilized Simulink for block-diagram modeling and automated code generation for deployment to flight controllers[cite: 38, 39].
  • Hardware Familiarity: Gained hands-on experience with the DJI F450 and Pixhawk 2.4.8 flight controller stack[cite: 27, 31].
  • Troubleshooting: Resolved complex parameter inconsistencies and hardware compatibility issues during the simulation-to-deployment pipeline[cite: 23, 24].

This research laid the foundation for future work in swarm robotics and autonomous navigation in GPS-denied environments[cite: 14, 46].

My Featured Project: Tello CV Drone Navigation

This project involved developing a real-time control system for a Tello drone. Using Python and the OpenCV library, I implemented computer vision techniques to enable two key features: Point-and-Click flying (where the drone navigates to user-selected coordinates) and ArUco marker tracking (allowing the drone to autonomously identify and follow a specific marker).
